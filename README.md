# Análise de Dados com SQL - Base Sakila

# Sobre o projeto
Este projeto tem como objetivo analisar o comportamento de clientes em uma locadora de filmes utilizando a base de dados Sakila.

Foram realizadas consultas SQL para extrair insights sobre:
- Clientes mais ativos
- Categorias mais alugadas
- Receita gerada por cliente
- Distribuição geográfica dos clientes

---

# Tecnologias utilizadas
- MySQL
- Base de dados Sakila
- SQL (JOIN, GROUP BY, agregações)

---

# Análises realizadas

## Total de locações por cliente
Identifica quais clientes mais realizaram aluguéis.

## Categorias mais alugadas
Mostra quais tipos de filmes são mais populares.

## Receita por cliente
Calcula quanto cada cliente gerou de receita.

## Clientes por cidade
Distribuição de clientes por localização.

---

## Principais Insights
- Alguns clientes possuem frequência muito maior de locações.
- Certas categorias dominam o volume de aluguéis.
- A receita está concentrada em um grupo menor de clientes.
- Há cidades com maior concentração de clientes ativos.
